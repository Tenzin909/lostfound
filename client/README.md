# Lost & Found Client

React frontend for the Lost and Found application built with Vite.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env` file (copy from `.env.example`):
   ```bash
   cp .env.example .env
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Features

- Browse lost and found items
- Filter by type, category, and search
- User authentication (login/register)
- Post new items (Lost or Found)
- View item details
- Claim found items
- Delete posted items
